# Torogi

**Torogi** is a mobile and web app dedicated to promoting **Igorot culture, history, and tourism**. It serves as a **community platform** for users to explore, share, and celebrate the rich heritage of the Cordillera region.  

---

## Features

- **Community Hub** – Connect with other users interested in Igorot culture.  
- **Events & Festivals** – Discover local cultural events and tourism activities.  
- **Multimedia Content** – View images and videos showcasing heritage, traditions, and tourist spots.  
- **Tourism Guides** – Access curated guides and tips for traveling within Cordillera.  
- **History & Culture** – Learn about Igorot history, traditions, and customs.  

---

## Tech Stack

- **Frontend:** HTML, CSS (Tailwind + custom styles), JavaScript  
- **Media:** Images (WebP), Videos (MP4)  
- **Deployment:** Netlify (free static hosting)
